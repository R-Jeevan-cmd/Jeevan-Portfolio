## ⚙️ Running Locally

Clone the repository:

bash git clone https://github.com/R-Jeevan-cmd/Jeevan-Portfolio.git 

Install dependencies:

bash npm install 

Start the development server:

bash npm run dev 

Build for production:

bash npm run build
